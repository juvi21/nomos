A function \(g:\mathbb{Z}^+\to\mathbb{Z}^+\) is said to be **stable** if
\[
g(x) \mid \bigl(y^{x} - g(y)^{\,g(x)}\bigr)
\]
for all positive integers \(x\) and \(y\).

Determine the smallest real constant \(\mu\) such that
\[
g(n) \le \mu n
\]
for all stable functions \(g\) and all positive integers \(n\).
